# Telegram Bot для автоматической рассылки сообщений

Бот для автоматической отправки сообщений с изображениями в Telegram через заданные интервалы.

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка

Отредактируйте `bot_config.json` или используйте веб-интерфейс после запуска.

### 3. Первый запуск (локально для авторизации)

```bash
python main.py
```

При первом запуске потребуется:
- Ввести код авторизации из Telegram
- После успешной авторизации будет создан `session_name.session` файл

### 4. Запуск бота

Откройте в браузере: http://localhost:5000

Через веб-интерфейс вы можете:
- Запустить/остановить бота
- Настроить все параметры
- Просмотреть статус

## Размещение на удаленном сервере

См. подробную инструкцию в файле `DEPLOY_BEGET.md`

**Важно:**
1. Сначала запустите бота локально для создания session файла
2. Загрузите `session_name.session` на сервер
3. Обновите пути в конфиге на относительные

## Структура проекта

```
telegramm_bot/
├── main.py              # Основной файл бота
├── bot_config.json      # Конфигурация
├── requirements.txt     # Зависимости
├── session_name.session # Файл сессии (создается при авторизации)
├── templates/
│   └── index.html      # Веб-интерфейс
└── image/
    └── cart_1.png      # Изображение для отправки
```

## Особенности

- ✅ Автоматическая отправка сообщений по расписанию
- ✅ Гибкое расписание (дни недели, часы, четные/нечетные даты)
- ✅ Веб-интерфейс для управления
- ✅ Защита веб-интерфейса паролем
- ✅ Поддержка отправки в группы и каналы
- ✅ Относительные пути (работает на любом сервере)
- ✅ Работает через ваш аккаунт (Telethon)
- ✅ Автозапуск через systemd или nohup

## Важные замечания

⚠️ **Бот использует ваш личный аккаунт Telegram** - соблюдайте лимиты Telegram, чтобы избежать блокировки.

Для постоянной работы 24/7 рекомендуется использовать VPS сервер.
