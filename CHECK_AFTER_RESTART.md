# ✅ Проверка после перезапуска

## Выполните эти команды:

```bash
cd /opt/reklama_bot

# 1. Проверить процесс
ps aux | grep "python3 main.py" | grep -v grep

# 2. Посмотреть последние логи
tail -50 bot.log

# 3. Проверить, что бот авторизован (должно быть "✅ Авторизация успешна!")
grep -i "авторизация\|authorized\|error" bot.log | tail -10
```

---

## Что искать в логах:

✅ **Хорошие признаки:**
- "✅ Авторизация успешна!"
- "⏳ Следующая отправка через..."
- Нет ошибок "database is locked"

❌ **Плохие признаки:**
- "❌ Критическая ошибка"
- "database is locked"
- "Требуется авторизация"
