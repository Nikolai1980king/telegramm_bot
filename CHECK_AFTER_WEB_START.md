# ✅ Проверка после запуска через веб-интерфейс

## После нажатия кнопки "Запустить бота":

Выполните на сервере:

```bash
cd /opt/reklama_bot

# Подождать 5 секунд после нажатия кнопки
sleep 5

# Проверить логи
tail -50 bot.log

# Проверить процесс
ps aux | grep "python3 main.py" | grep -v grep
```

---

## ✅ Что должно быть в логах:

- "Бот запущен"
- "✅ Авторизация успешна!"
- "⏳ Следующая отправка через X часов..."

---

## ❌ Если не работает:

1. Проверьте конфиг: `cat bot_config.json`
2. Проверьте session файл: `ls -lh *.session`
3. Проверьте ошибки в логах: `grep -i error bot.log | tail -10`
