# ⚡ Быстрый запуск бота

## На сервере выполните:

```bash
cd /opt/reklama_bot
python3 main.py
```

**Оставьте команду выполненной** - не нажимайте Ctrl+C, просто оставьте терминал открытым.

Вы увидите:
- Либо бот запустится и покажет "Running on http://..."
- Либо будут ошибки - тогда пришлите их мне

---

## Если бот запустился:

Нажмите **Ctrl+C** чтобы остановить, затем запустите через screen:

```bash
screen -S reklama_bot
python3 main.py
```

Затем **Ctrl+A**, затем **D**.

---

## Если есть ошибки:

Пришлите весь вывод команды `python3 main.py` - помогу исправить!
