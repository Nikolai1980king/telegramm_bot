# ⏰ Проверка времени сервера

## На сервере выполните:

### 1. Посмотреть текущее время:

```bash
date
```

### 2. Посмотреть часовой пояс:

```bash
timedatectl
```

Или:

```bash
cat /etc/timezone
```

---

## Важно:

- **Бот использует время сервера**, не вашего компьютера!
- Если сервер в UTC - время будет по UTC
- Если сервер в московском времени - будет по Москве
- При настройке расписания учитывайте время сервера!

---

## Как изменить часовой пояс (если нужно):

### Установить московское время:

```bash
timedatectl set-timezone Europe/Moscow
date
```

### Другие часовые пояса:

```bash
timedatectl list-timezones | grep -i moscow
timedatectl set-timezone Europe/Moscow
```

---

## Пример:

Если сервер в UTC (на 3 часа раньше Москвы):
- Вы настроите отправку на 12:00 (по UTC)
- В Москве это будет 15:00

**Решение:** Установите часовой пояс сервера на `Europe/Moscow`
